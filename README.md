## 📱 What this app does

motolink-android-intercom is an Android app for rider and pillion voice chat through helmets. It uses Bluetooth Classic for signaling, BLE for discovery, and WebRTC for voice on the same Wi-Fi network or hotspot.

Use it when you want a simple way to talk while riding without reaching for your phone.

## ✅ What you need

- An Android phone
- Bluetooth turned on
- Wi-Fi or a mobile hotspot for voice chat
- Two compatible phones if both riders want to use the app
- A helmet headset or earphones paired with the phone, if you use one

## 📶 How to use it

1. Turn on Bluetooth on both phones
2. Open the app on both devices
3. Let the phones find each other
4. Connect the rider and pillion devices
5. Join the same Wi-Fi network or hotspot
6. Start the voice link
7. Put on the helmet headset and test the sound

For the best result, keep both phones close during pairing and stay on the same network while talking.

## 🔧 First-time setup

### Pair Bluetooth

Bluetooth helps the app find nearby devices. Turn it on before opening the app. If asked to pair, approve the request on both phones.

### Connect to the same network

WebRTC voice works over shared Wi-Fi or a hotspot. Both phones should use the same connection before you start the call.

### Check audio

Make sure:
- The headset is paired
- Volume is turned up
- The microphone is not blocked
- The phone can play sound through the headset

## ❓ Common problems

### The app does not find the other phone

Check these items:
- Bluetooth is on for both phones
- Both phones are close together
- The app is open on both devices
- Battery saver is off if it limits Bluetooth scanning

### The voice does not work

Check:
- Both phones are on the same Wi-Fi or hotspot
- The headset is selected as the audio device
- The mic permission is allowed
- The volume is not muted

### The app will not install

Check:
- You downloaded the file from the releases page
- Your phone allows installs from unknown sources
- The file finished downloading before you opened it
